Projects

GUI Hangman Game

I have designed a GUI based simple Hangman Game using Python and Pygame. The game generates random words, and the users try to guess these random words letter by letter, and for every wrong letter guessed, the game adds a body part to the hanging man. The user gets 6 chances in total. This game uses Python modules like Pygame, Math, OS and Random.

The inspiration of this project comes from Tech with Tim's Pygame Tutorial.
